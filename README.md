# chatobizz

#### Objective:-

1. Made a chat room for freinds and families where users can connect with each other and share ideas,general talks.<br>

<h1>Usage:-</h1><br>

npm install<br>
npm run dev<br>

Go to localhost :3000<br>

  1.Node.js<br>
  2.Express.js<br>
  3.Socket.io<br>
  4.Javascript<br>
